Тема: Git и Github
Github - система управление версиями проектов (общих или личных) 

1) Репистрация на Github ---> https://github.com
2) скачать Git
3) Открыть терминал и проверить верси GIT 
           Git --version             #инициализация вашего проекта 
        I - команда: git init        #сохранение 
        II - команда: Git add
        III - компанда: git commit -m "Ваш коментарие к репозитори" # коментарие
